# EduPredict AI 🚀

AI-powered Student Performance Prediction System built as a **BCA 4th Semester Group Project**.

## Features

* Dynamic Excel / CSV upload
* Automatic subject detection
* ML-based CGPA prediction
* Risk level analysis
* Weak / Strong subject detection
* Personalized study hour allocation
* Smart timetable generation
* AI suggestions
* Class insights dashboard
* Student management (search / filter / delete)

## Tech Stack

* Python
* Flask
* MySQL
* HTML / CSS
* Chart.js
* Machine Learning
